## File System
: RAM에 저장된 데이터는 컴퓨터가 꺼지면 휘발되므로 컴퓨터가 꺼져도 데이터가 유지될 방법 필요  
: 운영 체제의 파일 시스템을 통해 데이터를 파일로 저장하고 관리  


**문제점**
- 데이터 종속성 문제 (운영 체제마다 파일 시스템이 다름)
- 데이터 중복성 문제
- 데이터 정확성 문제 (사용자가 파일에 직접 접근 가능)
- 데이터 일관성 문제 
- 데이터 보안성 문제
- 데이터 동시 접근 문제
